Welcome!

First, a few rules:

  • Include #editathon in your changeset/commit messages
  • Don't copy from other maps (unless you have permission)
  • Learn something new
  • Thinking about a data import? Tread carefully and read this first!
  • Have fun!

For beginners, here are some slides introducing OpenStreetMap.

Start editing now

Watch edits at Show Me the Way and OSM Live Map

Ideas for what to work on:

  • Pair up with someone new and teach them how to edit
  • Go outside and use a mobile app to add points of interest (POI) in the area. For example:
    • iOS: Pushpin
    • Android: Vespucci
  • Go through the iD editor walkthrough
  • Do a learnosm.org tutorial
  • Align streets and add POIs in your hometown
  • Help update OSM from new TIGER data with the Battlegrid
  • Help QA map data with KeepRight
  • Check out the Humanitarian OpenStreetMap Team task list

  • Interested in getting your data back out when you're done? Check this out

Remember, have fun!
